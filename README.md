# Metal Slug Clone — C++

A 2D side-scrolling action game built in C++ inspired by the classic Metal Slug series.
Built from scratch during 2nd semester at FAST University, Rawalpindi.

## Features
- Sprite-based animation using SFML
- Collision detection system
- Console-based notifications/debug output
- Enemy AI (Includes Boss AI as well)
- Terminal Screen Within Game For Dev Controls
- UI Made With SFML
- Terrain Generation Generative And Algorithm Based

## Concepts Applied
- Object-Oriented Programming (classes, inheritance)
- SFML graphics and audio integration
- Windows API usage (windows.h) 

## Requirements
- Windows OS
- SFML 2.x installed (https://www.sfml-dev.org/download.php)
- C++ compiler (MinGW / MSVC)

## How to Run
1. Install SFML and link it to your compiler
2. Clone this repo
3. Open in Visual Studio
4. Build and run the project

## Notes
Sprites and audio assets are included in the repository.
